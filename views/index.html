Exercise Tracker Project

Create a New User

Add exercises

POST /api/users/:_id/exercises

Once you add yourself as a user, you will see a response page in JSON form with an object containing your username and userId which you can use to add exercises.

You can also access a list of your exercise logs through querying the following:

GET user's exercise log: GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
